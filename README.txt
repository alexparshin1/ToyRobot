The ToyRobot can accept commands from:
- the command file: "ToyRobot commands.txt"
- the standard input: "ToyRobot" or "cat commands.txt | ToyRobot"

The unit tests are in the "UnitTests" directory.

The command files attached to this project are for the examples in the tests, as commands.txt and commands2.txt,
plus the one to test invalid commands, commands.invalid.txt.
